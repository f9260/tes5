VE3.Plus-Helper

VE3.Plus-Helper

VE3.Plus-Helper has been developed to make it easier for you to set your duration settings in the VE3.Plus system.

Thanks to this system, you can select the desired durations very easily, especially when you need a large number of duration settings. You can save them to your computer as a notebook with the download button or copy them directly and use them wherever you want.

Features

VE3.Plus-Helper can perform duration recordings from two different sources, Youtube and Local.

Youtube

You can use this option if you want to create duration settings for a video on YouTube.

Simply enter your Youtube link in the “Enter YouTube URL” field and click the button next to it.

Local

You can use the Local option when you want to perform operations with the videos you have on your computer.

Modes

When setting the duration settings, you can use Interval and Minute mode according to your needs in the VE3.Plus system.

Minute Mode

Using the Factory > Method > Minutes (opens in a new tab) method in the VE3.Plus system, You can directly select the durations you want to work.

The examples above are shown in Minute mode.

Minute/Range Mode

With the Factory > Method > Minute - Range (opens in a new tab) method, you can select all of the desired time intervals. Minute/Range mode helps you quickly specify the time intervals in which you want to perform work.

Interval Mode

When you want to use the Factory > Method > Interval (opens in a new tab) method, you can use the Interval Mode option.

Since only minute and hour intervals are specified in Interval Mode, only minute and hour intervals are specified when Interval Mode is enabled in our VE3.Plus-Helper system.
